# Student Course Management System 🎓
This is a C++ Console Application developed as an individual assignment to manage students and their course enrollments.

## Project Description
The system allows users to manage students through a menu-driven interface, applying core **OOP** principles and **STL** containers.

## Features
**Add Student:** Register with ID, Name, and GPA.
**Remove Student:** Delete record using Student ID.
**Search:** Find students by their unique ID.
**Enrollment:** Add courses to a student using `std::set` to avoid duplicates.
**Sorting:** Sort students by GPA from highest to lowest using STL `sort()`.

## Concepts Applied
**OOP:** Inheritance (Person -> Student), Encapsulation, and Polymorphism.
**STL:** `std::vector`, `std::set`, and `<algorithm>`.

## How to Run
1. Clone the repository.
2. Compile using: `g++ main.cpp -o student_app`.
3. Run the executable: `./student_app`.
